# Heart-disease-risk

# Installation:
```bash
pip install dagster dagit
```
# Create project:
```bash
dagster project scaffold --name heart_disease_risk
cd heart_disease_risk
```
# Execute:
```bash
make locally
```
# Open:
Open the following link in your browser in order to see the dagit UI:
```bash
http://127.0.0.1:3000
```
